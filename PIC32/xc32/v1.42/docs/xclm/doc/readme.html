1       Introduction

This is a general update to the Microchip MPLAB® XC Network Server. This version updates to RLM 11.2(BL2) and is the first version to be released as a separate installer.

1.1     Previous Versions

The previous MPLAB XC Network Server was version 1.23 and was release as part of the MPLAB XC Compilers produced after December 2013.

1.2     System Requirements

The MPLAB XC Network Server is available for a variety of operating systems, including Microsoft® Windows® XP Professional SP3/Vista SP1/Windows 7 Professional/Windows 8 and 8.1; Windows Server 2003, 2008, and 2012; Ubuntu® 9.10 and above; or Mac OS® X 10.5 and above. The compiler may also run on the various other Linux distributions, such as Oracle® Enterprise Linux® 5, Ubuntu 8.x and 10.04, Red Hat® Enterprise Linux. The 64-bit Linux® OS platforms are supported but you may need to install the 32-bit compatibility libraries for the Network Server to operate.

The MPLAB XC Network License Server does not operate on Virtual Machines.

1.3     Installation

Run the .exe (Windows), .run (Linux) or .app (OS X) compiler installer application, for example. XCNetServer-2.00.exe and follow the directions on the screen. Using the default installation directory is recommended. If you are using Linux, you must install the compiler using a terminal and from a root account. Install using a Mac OS X account with administrator privileges.

Full information on installation and operation of the MPLAB XC License Server is covered in the newest version of the XC License Server Manual (DS5002334).

2       What’s New

2.1     Version 2.00

  • Separate Installer - Previous versions of the MPLAB XC Network Server were installed by selecting the Server Install in one of the MPLAB XC Compiler installers. Starting with version 2.00, the Network Server has its own installer.
  • Updated to the newest version of RLM - MPLAB XC Network Server is based on the Reprise License Manager version 11.2(BL2).
  • Status messages include additional information - The status option for the included xclm command now displays additional information on when the HPA (High Priority Access) for the license expires.
  • New XC Network Server manual - This release includes a new manual that includes information on Installation and Operation of the MPLAB XC Network Server. Also included is a “How To” section that answers commonly asked questions.
  • Warnings are issued when HPA and demo licenses are about to expire - A warning is displayed when the license is within 14 days of the HPA or the demo license expiring.

3       Fixed Issues

3.1     Version 2.00

XCLM-77: Unable to Roam a license back into the server - If the Windows version of the rlm command is shut down with a license roamed out, then the license server may forget that the license is roamed out and may issue an error if an attempt to return the license occurs before it times out.

XCLM-78: Can’t read license data (-102) error issued when running any xlcm command - The Network Server installer that was included with each compiler was not configuring the xclm command. The new, separate installer used for v2.00 of the Network Server now configures the xclm command.

4       Known Issues

4.1     Version 2.00

XCLM-56: Host ID reported by 'xclm -hostinfo' changes on a same computer - If a removable network device (like an Android phone or a WiFi adapter) is recognized as the first MAC address found, it will display as the hostID when xclm -hostinfo runs. If the device is removed, and that hostID was used to activate a license, then the license will not be valid. xclm will always check each MAC address to see if it matches the hostID in the license file, so any fixed MAC address (with the '-'s removed) can be used as the hostID when the license is activated.
