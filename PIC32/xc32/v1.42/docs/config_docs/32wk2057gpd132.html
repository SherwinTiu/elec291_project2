Configuration-Bit Settings for PIC32WK2057GPD132

(Data Sheet # DS-)

Usage:
#pragma config SETTING = VALUE

TDOEN TDO enable for 2-wire JTAG
TDOEN = ON 2-wire JTAG protocol uses TDO
TDOEN = OFF 2-wire JTAG protocol does not use TDO
TROEN Trace Output Enable
TROEN = ON Start Trace Clock and enable Trace Outputs (Trace Probe must be present)
TROEN = OFF Stop Trace Clock and disable Trace Outputs
JTAGEN JTAG Enable
JTAGEN = ON JTAG Port Enabled
JTAGEN = OFF JTAG Port Disabled
USBSSEN USB Suspend Sleep Enable
USBSSEN = ON USB FREF clock is shut down when suspend mode is active
USBSSEN = OFF USB FREF clock continues to run when suspend mode is active
PGLOCK Permission Group LOCK
PGLOCK = ON CFGPG SFR bits are locked and cannot be modified
PGLOCK = OFF CFGPG SFR bits are not locked and can be modified
PMDLOCK Peripheral Module Disable (PMD) LOCK
PMDLOCK = ON PMDx SFR bits are locked and cannot be modified
PMDLOCK = OFF PMDx SFR bits are not locked and can be modified
IOLOCK IO LOCK
IOLOCK = ON IO Remap SFR bits are locked and cannot be modified
IOLOCK = OFF IO Remap SFR are not locked and can be modified
CFGLOCK Configuration Register Lock
CFGLOCK = ON All OTP memory self-writes, BCFG0 and System Config registers, are locked and can not be written - CFGLOCK value can not be changed
CFGLOCK = ON_W All OTP memory self-writes, BCFG0 and System Config registers, are locked and can not be written - CFGLOCK value can be changed
CFGLOCK = OFF All OTP memory self-writes, BCFG0 and System Config registers, are not locked and can be written - CFGLOCK value can be changed
OC_ACLK OCMP Alternate Clock Selection
OC_ACLK = OCMP_ALT_TMR OCMP module use an alternative TMR pair as their timebase clock
OC_ACLK = OCMP_TMR2_TMR3 All OCMP Modules use TMR2/TMR3 as their timebase clock
IC_ACLK ICAP Alternate Clock Selection
IC_ACLK = ICAP_ALT_TMR ICAP module use an alternative TMR pair as their timebase clock
IC_ACLK = ICAP_TMR2_TMR3 All ICAP Modules use TMR2/TMR3 as their timebase clock
EPGMCLK External Programming Clock enable
EPGMCLK = TSTFRC In ICSP (TMOD0) mode, enable the FRC clock to be driven by the TSTFRC Pin Function
EPGMCLK = FRC In ICSP (TMOD0) mode, the FRC clock is always driven by the FRC oscillator
DEBUG Background Debugger Access Selection
DEBUG = JTAG 4-wire JTAG I/F Enabled - EMUC/EMUD Disabled - ICD Module Disabled
DEBUG = JTAG_ICD 4-wire JTAG I/F Enabled - EMUC/EMUD Disabled - ICD Module Enabled
DEBUG = EMUC EMUC/EMUD Enabled - 4-wire JTAG I/F Disabled - ICD Module Disabled
DEBUG = EMUC_ICD EMUC/EMUD Enabled - 4-wire JTAG I/F Disabled - ICD Module Enabled
ICESEL EMUC/EMUD Communication Channel Select
ICESEL = ICS_PGx1 ICE EMUC1/EMUD1 pins are shared with PGC1/PGD1
ICESEL = ICS_PGx2 ICE EMUC2/EMUD2 pins are shared with PGC2/PGD2
ICESEL = ICS_PGx3 ICE EMUC3/EMUD3 pins are shared with PGC3/PGD3
ICESEL = ICS_PGx4 ICE EMUC4/EMUD4 pins are shared with PGC4/PGD4
TRCEN Trace Enable
TRCEN = ON Trace features in the CPU are enabled
TRCEN = OFF Trace features in the CPU are disabled
SMCLR Selects CRU handling of MCLR Control
SMCLR = MCLR_NORM Legacy mode (system clear does not reset all state of device)
SMCLR = MCLR_POR MCLR causes a faux POR
HSUARTEN UART1 High Speed Mode Enable
HSUARTEN = ON UART1 is driven from/to dedicated pins, resulting in the highest possible maximum baud rate
HSUARTEN = OFF UART1 is driven through PPS (I/O remap), resulting in a lower maximum baud rate
CLASSBDIS Disable CLASSB Device Functionality
CLASSBDIS = ENABLE CLASSB functions enabled
CLASSBDIS = DISABLE CLASSB functions disabled
USBIDIO USB USBID Selection bit
USBIDIO = ON USBID pin is controlled by the USB Module
USBIDIO = OFF USBID pin is controlled by the Port Function
